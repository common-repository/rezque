Book appointments from one to many users with custom settings from time zones, buffers, appointment time, start times, location addresses, and more!

== Description ==
Schedule appointments by allowing your customers to choose available slots based on multiple available resources along with getting appointment locations or setting an office location, or allowing the user to choose which one they prefer. 

Syncs with google calendars (must use google email).

[Find out more at https://rezque.com](https://rezque.com)

== Installation ==
Just upload the zipped file and sign up for an account at https://www.rezque.com/

Then add the shortcode [rezque id=""] and your calendar id to have your calendar rendered on your site.
